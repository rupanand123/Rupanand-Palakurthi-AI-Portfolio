## ⚙️ Technical Blueprint
- **Frontend Framework**: React 19 + TypeScript (Vite bundlers).
- **Styling system**: Tailwind CSS v4.
- **Server proxy**: Full-stack Node.js server with Express for secure server-side API proxying (hiding confidential key variables from browser contexts).
- **AI Integrations**: Native Google `@google/genai` TypeScript SDK.
- **Vercel Migration Ready**: Configured with a dedicated `vercel.json` rewrite strategy and dynamic `/api/chat` serverless backend route to run effortlessly on Vercel or standard Cloud Run platforms.

## 💻 Local Installation Controls

1. **Clone the matrix environment**:
   ```bash
   git clone https://github.com/rupanandpalakurthi/portfolio.git
   cd portfolio
   ```

2. **Supply Secret Parameters**: Create an `.env` configuration file in your workspace root directory and append your private client variables:
   ```env
   # Google Gemini API Key
   GEMINI_API_KEY="YOUR_GO_GEMINI_KEY"
   ```

3. **Install Core Dependencies**:
   ```bash
   npm install
   ```

4. **Launch Command Dev Server**:
   ```bash
   npm run dev
   ```

5. **Generate Production Bundle**:
   ```bash
   npm run build
   ```
